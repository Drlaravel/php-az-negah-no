### **فصل چهارم: آرایه‌ها در PHP**

---

### **4.1 مقدمه‌ای بر آرایه‌ها**

آرایه‌ها در PHP یک ساختار داده‌ای بسیار مفید و انعطاف‌پذیر هستند که به برنامه‌نویسان اجازه می‌دهند مجموعه‌ای از مقادیر را در یک متغیر ذخیره کنند. برخلاف متغیرهای معمولی که تنها یک مقدار را نگه‌داری می‌کنند، آرایه‌ها می‌توانند چندین مقدار از انواع مختلف داده‌ها (مانند اعداد، رشته‌ها، و حتی اشیاء) را در خود جای دهند. این قابلیت، آرایه‌ها را به یکی از مهم‌ترین ابزارهای برنامه‌نویسی در PHP تبدیل کرده است.

### **4.2 انواع آرایه‌ها در PHP**

در PHP، سه نوع اصلی آرایه وجود دارد که عبارتند از:

1. **آرایه‌های اندیسی (Indexed Arrays)**: در این نوع آرایه‌ها، مقادیر با استفاده از اندیس‌های عددی شناسایی می‌شوند.
2. **آرایه‌های انجمنی (Associative Arrays)**: در این نوع آرایه‌ها، هر مقدار با یک کلید رشته‌ای مرتبط است.
3. **آرایه‌های چندبعدی (Multidimensional Arrays)**: این نوع آرایه‌ها حاوی آرایه‌های دیگری به عنوان عناصر خود هستند.

#### **4.2.1 آرایه‌های اندیسی**

آرایه‌های اندیسی یکی از ساده‌ترین و پرکاربردترین انواع آرایه‌ها در PHP هستند. در این نوع آرایه‌ها، هر عنصر با یک اندیس عددی که از صفر شروع می‌شود، شناسایی می‌شود. به عنوان مثال:

```php
<?php
$fruits = array("Apple", "Banana", "Orange"); // تعریف یک آرایه اندیسی

echo $fruits[0]; // نمایش "Apple"
echo $fruits[1]; // نمایش "Banana"
?>
```

در این مثال، آرایه‌ای به نام `$fruits` ایجاد شده است که شامل سه مقدار است. این مقادیر با اندیس‌های 0، 1 و 2 قابل دسترسی هستند.

#### **4.2.2 آرایه‌های انجمنی**

آرایه‌های انجمنی به شما امکان می‌دهند که به جای استفاده از اندیس‌های عددی، از کلیدهای رشته‌ای برای دسترسی به مقادیر استفاده کنید. این نوع آرایه‌ها زمانی مفید هستند که نیاز دارید داده‌ها را با یک کلید مشخص که معنای خاصی دارد، ذخیره و بازیابی کنید. به عنوان مثال:

```php
<?php
$ages = array("Ali" => 25, "Maryam" => 30, "Hossein" => 35); // تعریف یک آرایه انجمنی

echo $ages["Ali"]; // نمایش 25
echo $ages["Maryam"]; // نمایش 30
?>
```

در اینجا، آرایه‌ای به نام `$ages` ایجاد شده است که سن افراد مختلف را با استفاده از نام آن‌ها به عنوان کلید ذخیره می‌کند. دسترسی به این مقادیر با استفاده از نام‌ها به جای اندیس‌های عددی صورت می‌گیرد.

#### **4.2.3 آرایه‌های چندبعدی**

آرایه‌های چندبعدی یکی از پیچیده‌ترین انواع آرایه‌ها در PHP هستند. این نوع آرایه‌ها می‌توانند حاوی آرایه‌های دیگری به عنوان عناصر خود باشند. این ویژگی به شما امکان می‌دهد تا ساختارهای داده‌ای پیچیده‌تری را مدیریت کنید. به عنوان مثال:

```php
<?php
$classes = array(
    "Ali" => array("Math" => 18, "Physics" => 17),
    "Maryam" => array("Math" => 20, "Physics" => 19)
); // تعریف یک آرایه چندبعدی

echo $classes["Ali"]["Math"]; // نمایش 18
echo $classes["Maryam"]["Physics"]; // نمایش 19
?>
```

در این مثال، آرایه‌ای به نام `$classes` ایجاد شده است که شامل نمرات دو دانش‌آموز در دو درس مختلف است. این ساختار داده‌ای به شما امکان می‌دهد تا داده‌های پیچیده‌تر را به صورت سازمان‌دهی‌شده ذخیره و مدیریت کنید.

### **4.3 عملیات مختلف بر روی آرایه‌ها**

کار با آرایه‌ها شامل عملیات مختلفی است، مانند افزودن، حذف، پیمایش و مرتب‌سازی عناصر. این عملیات به شما اجازه می‌دهد تا آرایه‌ها را به راحتی مدیریت و تغییر دهید.

#### **4.3.1 افزودن و حذف عناصر**

افزودن یک عنصر به آرایه، به سادگی با استفاده از علامت براکت‌ها (`[]`) انجام می‌شود:

```php
<?php
$fruits = array("Apple", "Banana");
$fruits[] = "Orange"; // افزودن "Orange" به انتهای آرایه
?>
```

همچنین، می‌توانید یک عنصر مشخص را از آرایه حذف کنید:

```php
<?php
unset($fruits[1]); // حذف عنصر با اندیس 1 (Banana)
?>
```

با این حال، پس از حذف یک عنصر، اندیس‌های سایر عناصر ثابت باقی می‌مانند و به‌صورت خودکار مرتب نمی‌شوند.

#### **4.3.2 پیمایش آرایه‌ها**

پیمایش آرایه‌ها معمولاً با استفاده از حلقه‌ها انجام می‌شود. یکی از متداول‌ترین روش‌ها برای پیمایش آرایه‌ها استفاده از حلقه `foreach` است:

```php
<?php
$fruits = array("Apple", "Banana", "Orange");

foreach($fruits as $fruit) {
    echo "Fruit: $fruit<br>";
}
?>
```

این حلقه هر عنصر را به ترتیب پیمایش می‌کند و نام میوه‌ها را نمایش می‌دهد.

برای پیمایش یک آرایه انجمنی نیز می‌توانید از `foreach` استفاده کنید:

```php
<?php
$ages = array("Ali" => 25, "Maryam" => 30, "Hossein" => 35);

foreach($ages as $name => $age) {
    echo "$name: $age<br>";
}
?>
```

در اینجا، کلیدها (نام‌ها) و مقادیر (سن‌ها) هر دو در حلقه دسترسی‌پذیر هستند.

#### **4.3.3 مرتب‌سازی آرایه‌ها**

PHP توابع متعددی برای مرتب‌سازی آرایه‌ها فراهم می‌کند. این توابع به شما امکان می‌دهند تا آرایه‌ها را به ترتیب صعودی یا نزولی مرتب کنید:

- استفاده از تابع `sort()` برای مرتب‌سازی صعودی:

  ```php
  <?php
  $numbers = array(4, 2, 8, 6);
  sort($numbers); // مرتب‌سازی به صورت صعودی
  ?>
  ```

- استفاده از تابع `rsort()` برای مرتب‌سازی نزولی:

  ```php
  <?php
  $numbers = array(4, 2, 8, 6);
  rsort($numbers); // مرتب‌سازی به صورت نزولی
  ?>
  ```

- استفاده از `asort()` و `ksort()` برای مرتب‌سازی آرایه‌های انجمنی بر اساس مقدار و کلید:

  ```php
  <?php
  $ages = array("Ali" => 25, "Maryam" => 30, "Hossein" => 35);
  asort($ages); // مرتب‌سازی بر اساس مقدار به صورت صعودی
  ksort($ages); // مرتب‌سازی بر اساس کلید به صورت صعودی
  ?>
  ```

### **4.4 توابع مفید برای کار با آرایه‌ها**

PHP مجموعه‌ای از توابع مفید برای کار با آرایه‌ها ارائه می‌دهد. در این بخش، برخی از پرکاربردترین این توابع معرفی می‌شوند:

- **`count()`**: تعداد عناصر موجود در آرایه را برمی‌گرداند.

  ```php
  <?php
  $fruits = array("Apple", "Banana", "Orange");
  $total = count($fruits); // تعداد عناصر در آرایه
  ?>
  ```

- **`array_merge()`**: دو یا چند آرایه را با هم ترکیب می‌کند.

  ```php
  <?php
  $array1 = array("Red", "Green");
  $array2 = array("Blue", "Yellow");
  $merged = array_merge($array1, $array2); // ترکیب دو آرایه
  ?>
  ```

- **`array_diff()`**: تفاوت بین دو یا چند آرایه را پیدا می‌کند.

  ```php
  <?php
  $array1 = array("Red", "Green", "Blue");
  $array2 = array("Green", "Yellow");
  $difference = array_diff($array1, $array2); // یافتن تفاوت بین دو آرایه
  ?>
  ```

- **`array_keys()`**: تمامی کلیدهای یک آرایه را برمی‌گرداند.

  ```php
  <?php
  $ages = array("Ali" => 25, "Maryam" => 30, "Hossein" => 35);
  $keys = array_keys($ages); //

 دریافت کلیدهای آرایه
  ?>
  ```

- **`array_values()`**: تمامی مقادیر یک آرایه را برمی‌گرداند.

  ```php
  <?php
  $ages = array("Ali" => 25, "Maryam" => 30, "Hossein" => 35);
  $values = array_values($ages); // دریافت مقادیر آرایه
  ?>
  ```

### **تمرین‌های فصل چهارم**

1. **ایجاد یک آرایه اندیسی:** یک آرایه از پنج عدد ایجاد کنید و با استفاده از `echo`، عنصر دوم آرایه را در مرورگر نمایش دهید.
2. **ایجاد یک آرایه انجمنی:** آرایه‌ای انجمنی ایجاد کنید که نام چند نفر و سن آن‌ها را ذخیره کند. سپس سن یکی از افراد را نمایش دهید.
3. **آرایه چندبعدی:** یک آرایه چندبعدی ایجاد کنید که اطلاعات چند دانش‌آموز و نمرات آن‌ها در دو درس ریاضی و علوم را ذخیره کند. نمره ریاضی یک دانش‌آموز مشخص را نمایش دهید.
4. **افزودن و حذف عناصر:** یک آرایه ایجاد کنید و چند عنصر به آن اضافه کنید. سپس یکی از عناصر را حذف کرده و آرایه نهایی را نمایش دهید.
5. **مرتب‌سازی آرایه:** یک آرایه از اعداد ایجاد کنید و آن را به صورت صعودی و نزولی مرتب کنید و نتیجه را نمایش دهید.
6. **پیمایش آرایه انجمنی:** یک آرایه انجمنی از نام و سن افراد ایجاد کنید و با استفاده از حلقه `foreach`، نام و سن هر فرد را نمایش دهید.

### **خلاصه فصل چهارم**

در این فصل، با آرایه‌ها در PHP آشنا شدید و نحوه تعریف، استفاده و مدیریت انواع مختلف آرایه‌ها را یاد گرفتید. انواع آرایه‌های اندیسی، انجمنی و چندبعدی مورد بررسی قرار گرفتند و شما یاد گرفتید چگونه عناصر را به آرایه‌ها اضافه و حذف کنید، آرایه‌ها را پیمایش و مرتب کنید. همچنین، توابع مفید برای کار با آرایه‌ها معرفی شدند که به شما کمک می‌کنند تا عملیات مختلفی را بر روی آرایه‌ها انجام دهید. این مهارت‌ها ابزارهای قدرتمندی برای مدیریت داده‌ها در برنامه‌های PHP به شما ارائه می‌دهند.
